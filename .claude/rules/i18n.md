# Internationalization (i18n)

All user-facing text in the application MUST be internationalized.

1.  **Never hardcode text strings** in components or pages.
2.  **Use `useTranslation` hook** from `react-i18next`.
3.  **Use `t('key')`** to display text.
4.  **Add new keys** to all supported languages (currently `en` and `fr`) in `i18n.ts`.
5.  **Group keys** logically (e.g., by screen or feature like `home`, `scan`, `edit`).
